# Screenshots

This folder stores UI screenshots referenced from the user and admin guides.

Examples of expected screenshots:

- `auth-page.png` – login / registration page
- `main-search-page.png` – main search page with filters
- `search-results.png` – search results table
- `select-datasets-for-collection.png` – selection of datasets
- `my-datasets-page.png` – user collections / history page
- `admin-main-page.png` – main admin section
- `admin-dataset-create-form.png` – dataset creation form
- `admin-requests-page.png` – monitoring of user requests / collections

In the Markdown docs we use placeholders like:

```md
<screenshot: auth-page>
```

After the UI stabilizes, replace these placeholders with real image links,
for example:

```md
![Auth page](../screenshots/auth-page.png)
```
