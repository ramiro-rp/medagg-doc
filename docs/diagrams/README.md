# Diagrams

Place visual diagrams related to the MedAgg project here.

Recommended types:

- **ER diagram** for the main database schema
- **High-level architecture diagram** showing backend, search module,
  frontend and storage
- Optional **sequence diagrams** for key flows:
  - search & build collection
  - admin editing dataset catalog

Preferred formats:

- `.drawio` / `.svg` / `.png`
- Name files with a prefix and short description, for example:
  - `er-medagg-core.svg`
  - `arch-medagg-mvp.png`
  - `seq-build-collection.drawio`

These diagrams can be referenced from Markdown docs using relative paths,
e.g.:

```md
![MedAgg architecture](../diagrams/arch-medagg-mvp.png)
```
